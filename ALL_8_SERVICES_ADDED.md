## 🔄 How It Works:

### On Page Load:
1. Fetch contacts from Google Sheets
2. Load default 8 services
3. Check for any custom services from contacts
4. Merge both lists
5. Display in grid

### When Customer Submits Form:
1. Contact saved to Google Sheets
2. Service extracted automatically
3. If new → Added to services list
4. If exists → Ignored (no duplicates)
