# ⚡ QUICK REFERENCE CARD - Admin Panel

## 🎯 **ESSENTIAL SHORTCUTS**

### **Navigation (Must Know!)**
```
Ctrl + D  →  Dashboard
Ctrl + O  →  Orders
Ctrl + C  →  Contacts
Ctrl + A  →  Analytics
Ctrl + ,  →  Settings
```

### **Actions**
```
Ctrl + K  →  Focus Search
Ctrl + R  →  Refresh Page
Ctrl + E  →  Export CSV
Shift + ? →  Show All Shortcuts
```

---

## 🔔 **TOP BAR ICONS** (Left to Right)

1. **🌙 Theme Toggle** - Switch light/dark mode
2. **🔔 Notification Bell** - Real-time alerts
3. **👤 User Profile** - Admin email display

---

## ⚡ **FLOATING BUTTON** (Bottom-Right)

**Big + Button** = Quick Actions Menu
- 🛒 New Order
- 👥 Add Contact  
- 📊 View Analytics
- 📄 Export Report
- ⚙️ Settings
- ✉️ Send Email

---

## 📊 **ORDERS PAGE FEATURES**

### **Top Buttons:**
1. **💰 Profit Calculator** - Calculate profits
2. **🔄 Refresh** - Reload orders
3. **📥 Export CSV** - Download data

### **Filters:**
- Search box (type anything)
- Status dropdown
- Date range (From/To)
- Sort by (Newest/Oldest/Amount)

### **Table Actions** (per row):
- **🕐 History** - Customer's all orders
- **👁️ View** - Order details modal
- **🖨️ Print** - Print invoice

### **Bulk Actions:**
1. Select orders with checkboxes
2. Blue bar appears at top
3. Choose: Print / Export / Delete

---

## 💰 **PROFIT CALCULATOR**

**Quick Steps:**
1. Click "Profit Calculator" button
2. Enter **Cost Per Service** (e.g., £50)
3. Enter **Fixed Costs** (e.g., £1000)
4. See instant results:
   - Total Revenue
   - Total Costs
   - Gross Profit (£)
   - Profit Margin (%)

---

## 👥 **CUSTOMER HISTORY**

**How to Check:**
1. Find customer in Orders table
2. Click **🕐 History** icon
3. See:
   - Total orders count
   - Completed vs Active
   - Total spent
   - All past orders timeline

---

## 🌙 **DARK MODE**

**Toggle Theme:**
1. Click theme icon (top-right)
2. Choose:
   - ☀️ Light
   - 🌙 Dark  
   - 💻 System (auto)

## 🔔 **NOTIFICATIONS**

**Manage Alerts:**
1. Click bell icon (top-right)
2. See all notifications
3. Click to mark as read

## 🚨 **TROUBLESHOOTING**

**Orders not showing?**
→ Click Refresh button or press Ctrl+R

**Can't find customer?**
→ Use search box with email/phone/name

**Export not working?**
→ Allow pop-ups in browser settings

**Dark mode not sticking?**
→ It's saved! Check localStorage or refresh

**Keyboard shortcuts not working?**
→ Make sure you're not typing in input field
