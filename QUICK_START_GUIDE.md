### **4. View Order Details**

**Steps:**
```
1. Go to Orders page
2. Find any order in table
3. Click 👁️ (eye icon) on right side
4. Beautiful modal opens with:
   - Order ID and date
   - Customer name, email, phone
   - Address
   - Services requested
   - Total amount
   - Status badge
5. Click "Print Invoice" from modal
6. Click "Close" when done
```

---

### **5. Print an Invoice**

**How to:**
```
Option 1: From Orders Table
- Click 🖨️ (printer icon) on order row
- Invoice opens in new window
- Press Ctrl+P to print

Option 2: From Order Details
- Open order details (click eye icon)
- Click "Print Invoice" button in modal
- Invoice prints beautifully
```

---

### **6. Export Orders to CSV**

**Simple steps:**
```
1. Go to Orders page
2. Apply filters if needed (optional)
3. Click "Export CSV" button at top
4. File downloads automatically
5. Open with Excel or Google Sheets
6. Done! 📊
```

## ❓ **TROUBLESHOOTING**

### **If something doesn't work:**

**Problem:** Orders not showing
```
Solution: 
- Check Google Sheets connection
- Verify sheet name is "Orders"
- Refresh page (Ctrl+R)
```

**Problem:** Export not working
```
Solution:
- Check browser pop-up blocker
- Allow popups for this site
- Try different browser
```

**Problem:** Keyboard shortcuts not working
```
Solution:
- Make sure you're not in input field
- Try pressing Shift+? for help
- Browser may need focus on page
```

**Problem:** Notifications not appearing
```
Solution:
- Check localStorage in browser dev tools
- Clear and refresh
- Notifications persist for 50 items max
```
